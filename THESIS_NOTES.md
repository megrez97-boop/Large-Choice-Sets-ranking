  - **Methodological Note (RBO vs Weighted Tau)**: While Weighted Tau can penalize top-rank errors, it requires arbitrary weight definitions and strict conjointness (lists must have the same length and identical elements), making it highly vulnerable to LLM hallucinations or dropped items. In contrast, RBO natively supports non-conjoint lists and uses a rigorous probabilistic parameter (`p`) to dictate top-heaviness (representing user persistence). This reinforces the theoretical justification for using RBO in top-heavy LLM ranking evaluations.

    - **Theoretical Review (Kendall's Tau vs RBO)**: 
      - **Kendall's Tau-B**: Evaluates overall structural consistency of ordinal rankings by comparing Concordant vs. Discordant pairs. Crucially, Tau-B handles "Ties" (同分), making it the standard metric in statistical packages. However, it treats all rank errors equally.
      - **RBO (Rank-Biased Overlap)**: Complements Tau by evaluating "Top-Heavy" accuracy. It heavily penalizes errors at the top of the list while forgiving noise at the bottom, making it ideal for LLM top-k recommendation tasks.
      - 📌 *For future deep-dives into Kendall's Tau variations (Tau-A, B, C) and manual calculation steps, refer to local file:* `library/Related paper/RBO and kendalls tau/statisticshowto.com-Kendalls Tau Kendall Rank Correlation Coefficient.pdf`
