# Megrez AI Master Suite 使用指南

這份指南配合 `START_AI_MASTER_SUITE.bat` 使用，專門為你的 RTX 3060 (12GB VRAM) 桌電打造，用於快速建置本機 AI 排序實驗環境。

## ⚠️ 執行前必讀 (重要！)

在點擊兩下執行腳本前，請確認以下事項，否則轉換 **31B 模型** 時極可能失敗：

1.  **NVIDIA 驅動程式更新**：請確保你的 RTX 3060 驅動程式是最新版的，否則腳本安裝 CUDA 時會發生衝突。
2.  **硬碟空間**：請確認你的 **E 盤** 至少有 **180 GB** 的空閒空間。
3.  **手動增加虛擬記憶體 (Pagefile)**：
    *   由於你的實體記憶體只有 16GB，轉換大型模型時會爆 RAM。
    *   請去 Windows 的 `進階系統設定` -> `效能` -> `進階` -> `虛擬記憶體`。
    *   取消自動分配，選擇 **E 盤**。
    *   將初始大小與最大值都設為 **`65536` MB (即 64GB)** 或更高。

## 🚀 如何使用腳本

1.  將 `START_AI_MASTER_SUITE.bat` 複製到你要執行的桌電上 (放在桌面或 E 盤皆可)。
2.  **連點兩下**執行它。
3.  它會自動彈出要求管理員權限的視窗，請點選「是」。
4.  在出現的選單中輸入 **`1`** 並按下 Enter，系統就會開始自動佈署。

### 佈署過程中可能會發生的事 (不用緊張)

*   **看似卡住**：在下載模型或轉檔時，畫面可能會停留在某個進度很久（尤其是轉檔 31B 模型時），這是正常的硬碟讀寫，請耐心等待，**千萬不要強行關閉**。
*   **紅字報錯後繼續**：如果它裝完某個工具後噴了紅字，請關閉腳本，**再連點兩下執行一次**。它會自動跳過已安裝的部分，從斷點繼續。

## 🧹 如何復原 / 卸載

如果你想清空所有安裝的工具並把 E 盤的幾十 GB 空間還回來：

1.  連點兩下執行 `START_AI_MASTER_SUITE.bat`。
2.  在選單中輸入 **`2`**。
3.  確認後，腳本會自動解除安裝 Python, Git, CMake, CUDA，並 **徹底刪除 E:\AI_Work 資料夾**。

## 🎉 成功後的下一步

當腳本顯示綠色的「佈署成功」後，你想要的 AI 模型就已經乖乖躺在 `E:\AI_Work\Models` 裡面了。
接下來，你就可以開啟 **Positron**，開始寫 R 腳本來呼叫這個位於本機的大腦，進行你的 Alpha Lattice 排序研究了！
