Each treatment i has n₁ = 16 first associates and these consist of the treatments which occur with it in the
same row, or in the same column, or correspond to the same letter from the first Latin square or correspond
to the same letter from the second Latin square. Thus the first associates of treatment 8 are:
6, 7, 9, 10; 3, 13, 18, 23; 2, 14, 20, 21; 4, 12, 16, 25.
The second associates of treatment i will be those treatments which are not its first associates. Thus the
n2=8 second associates of treatment 8 are 1, 5; 11, 15; 17, 19; 22, 24.
Further, any pair of treatments which are first associates will appear together in A, blocks and any pair
which are second associates will appear together in A₂ blocks of any design which is based on the L, (n)
type of association scheme.
